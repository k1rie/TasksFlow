# SmartClass - Sistema de Gestión Académica

🏆 **Ganador del Primer Lugar (Fase Estatal y Local) - Concurso Nacional de Prototipos DGETI 2025** 🏆

---

### 🚀 [Ver Demo en Vivo](https://smartclass.alphatech-labs.com/)

SmartClass es un sistema de gestión académica diseñado para simplificar y automatizar las tareas administrativas de los docentes. Permite llevar un control eficiente de calificaciones, asistencias vía QR y cálculos de promedios, todo en una plataforma web intuitiva.

Este proyecto fue desarrollado como parte del Concurso Nacional de Prototipos DGETI 2025, donde obtuvo el primer lugar a nivel estatal y local, destacando por su innovación y funcionalidad.

---

## ✨ Funcionalidades Principales

* **👥 Creación y Gestión de Grupos:**
    * Permite a los docentes crear y administrar múltiples grupos o clases.
    * Fácil inscripción y gestión de listas de alumnos por grupo.

* **📲 Asistencia con Códigos QR:**
    * Generación de códigos QR únicos y personales para cada alumno.
    * Sistema de escaneo rápido para tomar asistencia de forma eficiente, eliminando el pase de lista manual.
    * Registro automático de la fecha de la asistencia.

* **📊 Gestión de Calificaciones:**
    * Interfaz intuitiva para registrar las calificaciones de los alumnos por actividad.
    * Sistema flexible que se adapta a diferentes esquemas de evaluación.

* **📈 Cálculos y Reportes Automatizados:**
    * Cálculo automático de promedios de calificaciones por alumno y por grupo.
    * (Opcional: si lo tenías) Exportación de reportes en formato PDF o CSV.

* **👤 Perfiles para Docentes y Alumnos:**
    * Vistas personalizadas para docentes (control total).
